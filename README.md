# 🛠 My Fedora Live Setup

Быстрая настройка **Fedora Live USB** одной командой.

Скрипт автоматизирует базовую конфигурацию системы, чтобы не выполнять рутинные действия при каждом запуске Live-сессии.

---

## 📌 О проекте

`mysetup` — минималистичный bootstrap-скрипт для развертывания рабочего окружения в Fedora Live.

Подходит для:

* работы с Live USB
* аварийного восстановления
* быстрого развертывания среды на чужом ПК
* тестирования

---

## 🎯 Цели

* ⚡ **Скорость** — базовая настройка ~ за 1 минуту
* 🧩 **Минимализм** — только необходимые инструменты
* 🎛 **Интерактивность** — выбор компонентов при установке
* 🖥 **Чистый вывод** — понятный интерфейс в терминале

---

## 🚀 Запуск

В Fedora Live выполните:

```
curl -sSL https://raw.githubusercontent.com/Fantom1901/mysetup/main/run.sh | bash
```

Без клонирования репозитория — всё запускается одной командой.

---

## 📦 Что делает скрипт

### ⌨️ Настройка раскладки

* Добавляет русскую раскладку
* Переключение: Alt + Shift

### 📝 Установка CLI-инструментов

* `micro`
* `jq`

### ✈️ Быстрый доступ к Telegram Web

* Удобный запуск веб-версии Telegram

### 🛡 Установка VPN

* Автоматическое скачивание и установка последнего релиза Amnezia VPN

### 🤖 Дополнительные инструменты

* Установка утилиты `gptcopy`

---

## ⚙️ Как это работает

* Использует стандартные инструменты Fedora (`dnf`, `curl`)
* Загружает актуальные релизы напрямую из GitHub
* Работает в рамках Live-сессии (без перезагрузки)

---

## 📌 Требования

* Fedora Live (актуальная версия)
* Подключение к интернету
* Права `sudo`
